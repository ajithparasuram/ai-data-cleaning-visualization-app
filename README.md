AI Data Cleaner & Visualizer

An interactive web-based application built with Python and Streamlit to automate data cleaning and simplify data analysis workflows.

🚀 Live Demo

https://s7rflrtuhnne4k8lut7zsy.streamlit.app/

---

📌 Overview

In many real-world scenarios, data preparation takes more time than actual analysis. This application was designed to reduce manual effort by automating common data preprocessing tasks and making datasets instantly usable for further analysis.

The tool allows users to upload raw datasets and transform them into structured, analysis-ready data with minimal effort.

---

⚙️ Key Features

- Upload CSV or Excel files
- Automated data cleaning pipeline
- Missing value handling (Median/Mode or KNN Imputation)
- Outlier detection using Isolation Forest
- Interactive data visualization
- Download cleaned dataset
- Simple and user-friendly interface

---

🧠 Use Case

This application was developed to streamline internal data workflows and reduce repetitive manual preprocessing. It helps both technical and non-technical users quickly prepare data for reporting and analysis.

---

🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit

---

🔄 Workflow

1. Upload raw dataset (CSV/Excel)
2. Select cleaning options
3. Apply automated transformations
4. Visualize processed data
5. Download cleaned dataset
